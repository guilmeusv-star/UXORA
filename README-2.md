## Comment activer GitHub Pages

Pour que le lien ci-dessus fonctionne, suis ces étapes :

1. Va sur ton dépôt GitHub
2. Clique sur **Settings** (Paramètres)
3. Dans le menu gauche, clique sur **Pages**
4. Sous *Branch*, sélectionne **main** puis **/root**
5. Clique sur **Save**
6. Attends 1-2 minutes → ton site est en ligne !

Ton lien final sera :
```
https://ton-username.github.io/ton-repo/
```
